# Data Analysis / Programming Practice

## 概要
本リポジトリは、Pythonを用いたデータ分析・数値計算・機械学習の基礎を学習するために作成したものです。

主に、統計分析や機械学習の基本的な処理を、ライブラリに依存しすぎず自分で関数として実装することで、各手法の仕組みを理解することを目的としています。

## 学習目的
コンサルティング業務において、データに基づいて課題を構造化し、仮説検証を行うための基礎的な分析力を身につけることを目的としています。

## 内容
- 基本的な統計量の計算
- 線形回帰
- ロジスティック回帰
- 勾配降下法
- 行列計算
- データの前処理
- 可視化の練習

## 工夫した点
単に既存ライブラリを使うだけでなく、回帰分析や勾配降下法などの処理を自分で関数として実装し、アルゴリズムの内部構造を理解することを重視しました。

また、コードを機能ごとに整理し、後から分析課題に応用しやすい形で管理しています。

## 使用技術
- Python
- NumPy
- pandas
- matplotlib
- scikit-learn
